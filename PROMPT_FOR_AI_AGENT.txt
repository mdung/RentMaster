You are a senior UI/UX designer tasked with creating a complete design system and interface designs for RentMaster, a room rental management web application.

PROJECT OVERVIEW:
RentMaster is a React + TypeScript web application for property managers to manage properties, rooms, tenants, rental contracts, invoices, and payments. The current UI is basic and needs a complete modern redesign.

CURRENT STATE:
- Basic sidebar navigation + main content layout
- 7 pages: Login, Dashboard, Properties & Rooms, Tenants, Contracts, Invoices, Payments
- Minimal styling, no design system
- Dark sidebar (#2c3e50), white content area
- Basic tables, forms, modals without modern styling

DESIGN REQUIREMENTS:

STYLE & AESTHETIC:
- Modern SaaS application style (similar to Stripe Dashboard, Linear, or Notion)
- Clean, professional, minimal but data-dense
- Color palette: Professional blue primary (#3B82F6 or #2563EB), semantic colors (green success, amber warning, red error), neutral grays
- Typography: Modern sans-serif (Inter or system fonts), clear hierarchy
- Spacing: Consistent 4px/8px scale
- Components: Modern cards with shadows, clean tables, polished forms

PAGES TO DESIGN:

1. LOGIN PAGE
- Centered login form with brand
- Error message display
- Loading state on submit
- Clean, minimal design

2. DASHBOARD PAGE
- Grid of 6-8 stat cards showing: Total Rooms, Occupied, Available, Maintenance, Active Contracts, Monthly Revenue, Total Outstanding
- Charts section: Revenue trend (line chart), Occupancy rate (bar/pie), Payment status distribution
- Recent activity feed
- Upcoming due dates list
- Quick action buttons

3. PROPERTIES & ROOMS PAGE
- Property cards in grid layout (name, address, room count, actions)
- When property selected: Rooms table (Code, Floor, Type, Status badge, Base Rent, Actions)
- Add Property/Room buttons
- Search and filter bar
- Modal forms for create/edit

4. TENANTS PAGE
- Data table: Name, Phone, Email, ID Number, Address, Actions
- Add Tenant button
- Search and filter
- Tenant detail modal

5. CONTRACTS PAGE
- Table: Code, Room, Tenant, Start/End Date, Rent Amount, Status badge, Actions
- Add Contract button
- Filter by status, property, date
- Comprehensive contract form modal

6. INVOICES PAGE
- Table: Invoice #, Contract, Tenant, Period, Amount, Status badge, Due Date, Actions
- Generate Invoice button (prominent)
- Filter by status, date range
- Invoice detail view with items and payments
- PDF export button (if applicable)

7. PAYMENTS PAGE
- Table: Payment #, Invoice, Amount, Date, Method, Actions
- Add Payment button
- Filter by invoice, date
- Payment form modal

  - Room: Available (green), Occupied (blue), Maintenance (orange)

CARDS:
- Property cards, invoice cards, contract cards
